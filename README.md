![tool](https://img.shields.io/badge/tool-xcode6-blue.svg)
![language](https://img.shields.io/badge/language-swift2-yellow.svg)

### 概要
- Swift勉強用サンプルソース

### このアプリについて
- 韓国語を漫画で勉強できる
- 状況ごとの韓国語レッスンができる
- キャラクターごとの音声も提供
