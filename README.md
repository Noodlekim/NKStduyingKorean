# NKStduyingKorean

### 環境
- Xcode7で起動
- ~~Swift1.1~~ Swift2

### 概要
- Swift勉強用サンプルソース

### このアプリについて
- 韓国語を漫画で勉強できる
- 状況ごとの韓国語レッスンができる
- キャラクターごとの音声も提供
